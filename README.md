# FileServiceAllinOne
一个将 FTF、SMB、NFS 三种文件协议一键启动的软件
本软件主要以 Python 语言编写出，起作用就是调用 Windows CMD 中的 DISM 命令来启用 Windows 功能的软件
一键开启服务 一键关闭服务 无需进行过多的配置。

# 测试环境
Windows 11 家庭版（笔记本电脑 没搞专业版）
